Projetos

FisioData

O FisioData é uma aplicação web desenvolvida em Angular e hospedada no Firebase, voltada para a gestão eficiente de dados na fisioterapia. A plataforma permite o cadastro de pacientes e a geração automatizada de relatórios personalizados, otimizando o acompanhamento dos tratamentos e a tomada de decisões estratégicas.

Buguer do Chefão

Burger do Chefão é um site responsivo desenvolvido com HTML, CSS e JavaScript, focado na apresentação de produtos e na geração de pedidos via WhatsApp para uma hamburgueria. O projeto destaca boas práticas de desenvolvimento web, design responsivo e integração de funcionalidades que facilitam o contato e aumentam as conversões.
